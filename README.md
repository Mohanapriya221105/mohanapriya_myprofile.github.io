# Personal Portfolio Website

This is a responsive, static personal portfolio website created using HTML5 and CSS3.

## Features
- **Clean & Modern Design**: Blue and Gray color theme with professional typography.
- **Responsive Layout**: Mobile-first design using Flexbox and Grid.
- **CSS-only Components**:
  - Image Slider (Home Page) using `@keyframes`.
  - Responsive Mobile Menu (using checkbox hack).
- **Sections**:
  - Home: Welcome message and image slider.
  - About: Profile information, education, and career objective.
  - Skills: Grid layout of technical skills.
  - Projects: Showcase of projects with descriptions.
  - Contact: Functional-looking contact form and details.

## How to Run
Simply open `index.html` in any modern web browser.

## Customization
- **Images**: Replace files in `assets/images/` with your own images (JPG/PNG).
- **Colors**: Edit the `--primary-color` and `--secondary-color` variables in `css/style.css`.
- **Content**: Update the HTML files with your personal details.
